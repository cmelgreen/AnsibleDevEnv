# AnsibleDevEnv
Setup development environment using Ansible

Copy-paste to set up a new development environment:

`git clone https://github.com/cmelgreen/AnsibleDevEnv && cd AnsibleDevEnv && bash bootstrap.sh`

**Includes:**

**Languages:**

- Python
- Go
- NodeJS
- React

**Database:**

- Postgres
- PGAdmin4

**DevOps:**

- Docker
- Terraform
- Pakcer
- Jenkins
- Bazel
